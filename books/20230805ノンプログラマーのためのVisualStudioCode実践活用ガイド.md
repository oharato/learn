# 拡張機能
- zenkaku
- code-eol 改行記号
- CharacterCount 文字数カウント
- Japanese Word Handler カーソル操作を日本語向き Ctrl + →
- novel-writer 小説執筆
- テキスト校正くん
- Marp for VS Code Markdownファイルからスライド
- CSS Peek HTMLファイル編集中にCSS定義を確認
- Image preview HTMLファイル編集中にファイルパスにマウスオーバーでプレビュー

# 操作
- 選択した同じテキストを全選択 ctrl + shift + L
- 選択した行を複製 Alt + Shift + ↓
- カーソルを上下に拡大 Ctrl + Alt + ↑↓
- 矩形選択 Shift + Alt
